# Chat on Flask

A simple chat application built with Flask.

## Technologies Used

- **Flask**: Web framework for Python.
- **SQLite3**: Database for storing users, chats, and messages (built-in Python standard library).
- **HTML/CSS**: Frontend templates and styling.
- **Jinja2**: Templating engine (included with Flask).

## Features Implemented

- User registration and login.
- Database initialization with tables for Users, Chats, Users_Chats, and Messages.
- Basic chat interface displaying list of chats with last messages.
- Static files for CSS styling.
